1.3 RESUMEN
Este componente conecta EMatches a una base de datos, configurada a traves del componente <C-98>
Este componente conoce como hacer CRUD de Instancias de EMatch en cualquier base de datos configurada

Este componente debe ser ensamblado en DeltaJ para producir un objeto .Java
